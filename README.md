# buxPHP

# Introduction
BuxPHP is a simple PHP library that will help you implement [Bux.ph](https://bux.ph) payment in your projects.

# Features
- "Creating Payment Request" / "Generate Payment Link"
- Get payment info
- Validate IPN message
